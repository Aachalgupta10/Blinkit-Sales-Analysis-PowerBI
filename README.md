# Blinkit Sales Analysis Dashboard

This project analyzes Blinkit grocery sales data using Power BI.

## Tools Used
- Power BI
- Excel
- Data Visualization

## Dashboard Features
- Total Sales KPI
- Average Sales
- Item Category Performance
- Outlet Location Analysis
- Fat Content Distribution
- Sales Trend by Year

## Key Insights
- Identified top selling product categories
- Analyzed sales distribution across outlet sizes
- Observed sales growth trends by outlet establishment year
